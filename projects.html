SpaceX Launch Data

By webscraping the Wikipedia page for SpaceX launch data and accessing the SpaceX API, I created an exhaustive database of SpaceX launch data. After cleaning the data and building a dash app, I trained several machine learning models to predict landing success of Falcon 9 launch missions. My most successful model was a decison tree with a success rate of 81%. This was the final project for a Coursera certificate. The scripts and source code can be found here.


Airline Peformance and Delay Data

Using a pandas dataframe of airline data, I created a dash app that displays data and statistic for the performance and delays of various airlines from 2005 to 2019. This was the final project for a Coursera course. The source code can be found here.
